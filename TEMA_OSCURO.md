#  Tema Oscuro - Generador de Addenda Soriana

##  ¡Nueva Interfaz Modo Oscuro Implementada!

###  **Características del Tema Oscuro:**

- **Colores Suaves**: Paleta de colores oscuros que no lastiman la vista
- **Gradientes Modernos**: Botones con gradientes elegantes
- **Animaciones Sutiles**: Transiciones suaves y efectos hover
- **Responsive**: Se adapta perfectamente a cualquier dispositivo
- **Alternancia de Tema**: Botón flotante para cambiar entre claro/oscuro

### **Paleta de Colores:**

```css
/* Fondos */
--bg-primary: #0d1117    /* Fondo principal */
--bg-secondary: #161b22  /* Fondo secundario */
--bg-card: #1c2128       /* Fondo de tarjetas */

/* Textos */
--text-primary: #e6edf3  /* Texto principal */
--text-secondary: #8b949e /* Texto secundario */

/* Acentos */
--accent-blue: #58a6ff   /* Azul principal */
--accent-green: #56d364  /* Verde para éxito */
--accent-red: #f85149    /* Rojo para errores */
```

###  **Nuevas Funcionalidades:**

1. **Botón de Alternancia**: Esquina superior derecha
2. **Estados Visuales**: Validación en tiempo real con colores
3. **Iconos en Tarimas**: Emojis descriptivos
4. **Gradientes**: Botones y elementos con gradientes modernos
5. **Persistencia**: Recuerda tu preferencia de tema
